# Text Analysis Project

This project contains two Jupyter notebooks:



## Part 1: Text Analysis
`02_text_analysis.ipynb`

This notebook includes:
- text preprocessing
- sentiment analysis
- TF-IDF vectorization
- clustering
- similarity matrix visualization
- PCA visualization

## Part 2: Image to PDF Conversion
`01_image_to_pdf.ipynb`

This notebook:
- reads images from an `images/` folder
- converts them to RGB when needed
- updates image DPI metadata to 1200
- saves PDF versions into `journal_images/`
